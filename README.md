## ripper

Get licenses from files.

Currently, supports ripping out head paths of the Apache License formatted with

// Copyright ...

# Installation:

```shell
$ go get github.com/odeke-em/rippper
```

# Usage

```shell
$ ripper [packages....]
```

e.g

```shell
$ ripper github.com/odeke-em/drive/src github.com/github/hub
```
